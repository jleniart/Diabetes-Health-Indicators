# Diabetes-Health-Indicators
This was a group project where we analyzed a dataset with health metrics and developed classification models to predict whether or not a patient was diabetic. 

The dataset we used was published on Kaggle: https://www.kaggle.com/datasets/alexteboul/diabetes-health-indicators-dataset?select=diabetes_binary_5050split_health_indicators_BRFSS2015.csv

The elements of this project that I worked on individually were: Exploratory Data Analysis, Random Forest model development, and compiling all model results.
